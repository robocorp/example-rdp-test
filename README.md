# example-rdp-test

Contains Robocorp Robot which only tests the Windows RDP connection functionality.

## Robot steps
  
  1. Open Windows calculator
  2. Input number `123`
  3. Take Screenshot
  4. Teardown
  5. Screenshot either `success.png` or `failure.png`
